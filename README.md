# How to use GitHub Basic:

# Step 1: Create a repository:
1.1-->In the upper-right corner of any page, select , then click New repository.
1.2-->In the "Repository name" box, type repository name (hello-world).
1.3-->In the "Description" box, type a short description. For example, type "This repository is for practicing the GitHub Flow."
1.4-->Select whether your repository will be Public or Private.
1.5-->Select Add a README file.
1.6-->Click Create repository.

# Step 2: Create a branch:
2.1-->Creating a branch of existing repo:
2.2-->Click the <>Code tab of your hello-world repository.
2.3-->Above the file list, click the dropdown menu that says main.
2.4-->Type a branch name, readme-edits, into the text box.
2.5-->Click Create branch: readme-edits from main.


# Step 3: Make and commit changes:
3.1-->Under the readme-edits branch you created, click the README.md file.
3.2-->To edit the file, click .
3.3-->In the editor, write a bit about yourself.
3.4-->Click Commit changes.
3.5-->In the "Commit changes" box, write a commit message that describes your changes.
3.6-->Click Commit changes.

# Step 4: Open a pull request:
4.1-->Click the Pull requests tab of your hello-world repository.
4.2-->Click New pull request.
4.3-->In the Example Comparisons box, select the branch you made, readme-edits, to compare with main (the original).
4.4-->Look over your changes in the diffs on the Compare page, make sure they're what you want to submit.
4.5-->Click Create pull request.
4.6-->Give your pull request a title and write a brief description of your changes. You can include emojis and drag and drop images and gifs.
4.7-->Click Create pull request.

# Step 5: Merge your pull request:
5.1-->At the bottom of the pull request, click Merge pull request to merge the changes into main.
5.2-->Click Confirm merge. You will receive a message that the request was successfully merged and the request was closed.
5.3-->Click Delete branch. Now that your pull request is merged and your changes are on main, you can safely delete the readme-edits branch. If you want to make more changes to your project, you can always create a new branch and repeat this process.
5.4-->Click back to the Code tab of your hello-world repository to see your published changes on main.
